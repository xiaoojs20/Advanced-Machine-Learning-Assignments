# 微调作业: LLM 风格迁移与对齐

基于 LoRA 技术的参数高效微调 (PEFT) 实验，使大型语言模型能够适应特定的情感基调与互联网语体。

## 项目结构
- **src/**: 包含数据处理逻辑及主要的微调分析代码。
- **data/**: 用于风格迁移训练的精选数据集。
- **微调结果/**: 包含微调前后模型表现的对比截图与量化指标。

## 实验目标
- **语体识别与迁移**: 训练模型掌握特定的互联网俚语及文化语境（如“B站风格”）。
- **人格对齐**: 使模型输出能够与特定的性格特征及用户情感状态保持一致。

## 核心技术
- **LoRA (低秩自适应)**: 在冻结原始模型参数的基础上，仅更新极小比例的权重，实现高效适配。
- **PEFT 框架**: 利用 Hugging Face 的 PEFT 库进行适配器管理与部署。
